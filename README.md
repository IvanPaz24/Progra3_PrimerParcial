# 🎬 Buscador de Series

Aplicación web que consume una API externa para buscar y mostrar series en pantalla, con funcionalidad para guardarlas como favoritas.

## Tecnologías
- HTML, CSS, JavaScript (vanilla)
- Fetch API / consumo de API REST

## Funcionalidades
- Búsqueda de series en tiempo real
- Visualización de resultados con póster y descripción
- Guardado de series favoritas en almacenamiento local
